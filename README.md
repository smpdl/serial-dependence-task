# COGS-219 Research Methods in Cognitive Science

## Project Title
Serial Dependence Task

## Description
This study tests whether serial dependence is a single global mechanism or consists of multiple independent processes by examining individual differences across three domains: orientation, color, and numerosity. We predict that if it is global, individuals will show stable effect strengths across all tasks; if it is feature-specific, these effects will vary across domains. Participants complete a within-subjects task where they judge sequential stimuli and adjust responses using a slider, and serial dependence is measured as the bias toward the previous trial’s stimulus. We compare effect sizes across conditions while controlling for spatial location, timing, and attention to assess cross-domain stability.
